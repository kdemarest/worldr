# **Session Reminders**

At Session Start

* Players: pursue your Character **Arcs**  
* Players: get creative with Fate-burning **Intrusions**  
* Players: use **Cyphers** like CoH inspirations  
* GM: hand out Fate for role playing and in-character successes or failures  
* GM: hand out Cyphers like candy

At Session Completion

* 2-3 exp generally per player for Arcs  
* Any new Character Arcs starting? Ending?  
* Allocate any new contacts, enemies, favors owed, debts of honor  
* Significant changes in social standing  
* Review whether infection, disease, etc might set in
