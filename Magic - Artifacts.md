# Artifacts

Some objects or places have a spirit tethered to them.

## **Creating** {#creating-1}

* Follows the Cypher rules for Crafting (pg 228), difficulty 5+Level  
* Requires rare knowledge  
* Costs Int equal to Level forever (not regained when the artifact is destroyed)

## **Abilities** {#abilities-1}

Spirits tethered to artifacts might:

* Perform only a single specific magic  
* Allow any use of the artifact's tethered spirit  
* Empower spirits tethered to the mage to do more

The following abilities are allowed as artifacts only:

* Abilities related to machine enhancement, or powered armor  
* Machine Enhancement (e)   
* T1 Heads-Up Display (i2+)
